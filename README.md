# Gamified AI Learning Platform

## Overview
This project is a gamified AI-supported learning platform designed to support perceived knowledge retention in digital learning environments. The platform includes points, levels, badges, challenges, progress tracking, immediate feedback, and rule-based adaptive behavior.

## Features
- Lessons and quizzes
- Immediate feedback
- Points, levels, and badges
- Daily challenges
- Progress tracking
- Rule-based adaptive hints and difficulty
- Survey reminder for evaluation

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- Local Storage

## How to Run
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the project using `npm run dev`.
